Hi! I am Akash Yadav, a BCA student and aspiring MERN Stack Developer. This is my very first hands-on web development mini-project built from scratch after breaking out of tutorial hell!

## 🛠️ Features (Kya-Kya Functions Hain)
- **Dark Mode UI:** Sleek black background with high-contrast text for a modern developer look.
- **Interactive GIF:** A cool coding GIF embedded at the top of the card.
- **Dynamic Follow Button:** Built with JavaScript to toggle between "Follow Akash" (Green) and "Following ✓" (Blue-Green) smoothly upon clicking.
- **Hover Effects:** Smooth zoom-in scale effect on the main CTA button.

---

## 💻 Tech Stack Used (Kis cheez se bana hai)
- **HTML5:** For semantic project structure.
- **CSS3:** For styling, including **Flexbox** (for absolute screen centering), transitions, and custom colors.
- **Vanilla JavaScript:** For DOM manipulation and handling user click events.

---

## 🧠 What I Learned From This Project
1. **The Box Model:** Understood the practical difference between padding (internal spacing) and margin (external spacing).
2. **Flexbox Superpower:** Learned how to center any card perfectly on the screen using just 3 lines of CSS on the parent element.
3. **Event Listeners:** Mastered how to capture a `click` event in JS and dynamically change text (`innerText`) and styles (`style.backgroundColor`).

## ⚙️ How to Run This Locally
If you want to run this project on your machine:
1. Clone this repository or download the files.
